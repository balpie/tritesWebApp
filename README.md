# tritesWebApp
Trites, a game that probably will remind you of an other one very similar...
- Project written for Progettazione Web course @unipi

## Comandi
- Invio: hard drop (+ 10pts * liv)
- Right Shift: hold
- W: ruota
- A, D: sposta a sinistra, destra
- S: Scendi veloce
- Pausa: Ferma il clock del computer
